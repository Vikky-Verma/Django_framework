=> for creating virtual environment
python -m venv .venv -> on windows

=> for activate virtual environment
.venv\Scripts\activate  -> on windows
source .venv/bin/activate   -> on macOs and linux

=> for deactivate virtual environment
deactivate

=> To install django
pip install Django

=>create a django Project
django-admin startproject projectname

=>To run the development server
python manage.py runserver

=>django flow structure
User ---> request -->urls.py --->views.py --->response --> user


=> To create app 
python manage.py startapp appname
                                    


=> To download django tailwind 
pip install django-tailwind 
pip instll 'django-tailwind[reload]'

=> To initialize tailwind 
python manage.py tailwind init

=> To install tailwind
python manage.py tailwind install

=> to tailwind run in django  in different terminal
python manage.py tailwind start

=> To migrate 
python manage.py migrate

=> To create superuser
python manage.py createsuperuser
python manage.py runserver
